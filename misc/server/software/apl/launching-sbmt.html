There are several ways to launch the SBMT without using a web browser. In all cases, you will be prompted for your username and password when outside the lab. In addition, in all cases, the latest available version of the tool is always run.

Command Line (All platforms)

The SBMT can be started from the command line with the following command: 

javaws http://near.jhuapl.edu/software/apl/sbmt/near.jnlp

Mac Application Bundle (Mac only)

Download the file sbmt-launcher.zip and unzip it to any directory. From within Finder, you can then double click on the SBMT Launcher application to launch the tool. In addition, you can drag the application to the Dock and launch it from there.
